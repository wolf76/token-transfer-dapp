# Token Transfer DApp

Polygon Token Transfer DApp PoC.
Created using VueJS and Polygon's matic.js

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
